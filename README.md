# 🌟 Certiverse – AI-Powered Assessment and Credentialing Platform

Certiverse is a modern full-stack freemium assessment platform that allows users to take quizzes for free, unlock their results through a payment gateway flow, and receive an auto-generated professional PDF certificate.

## 🚀 Live Demo

### Frontend

https://certiverse-credential-platform-fron.vercel.app/

### Backend API

https://name-certiverse-backend.onrender.com

### GitHub Repository

https://github.com/Vighneshvikki/certiverse-credential-platform

---

# ✨ Features

* 🎯 Category-based skill assessments
* 📚 Multiple professional domains
* ⏱ Timed quizzes
* 📈 Automated score calculation
* 🔒 Freemium paywall model
* 💳 Payment gateway architecture (Razorpay compatible)
* 🎉 Payment success flow
* 📊 Detailed performance results
* 📜 Professional PDF certificate generation
* 🌙 Modern UI with dark mode
* 📱 Fully responsive design
* 🚀 Live deployment
